# Автоматизация тестирования API сервиса заказов

Автотесты для проверки работы API сервиса заказов.

## Сценарий тестирования
- Создание нового заказа
- Получение данных заказа по треку
- Проверка успешности операций

## Установка зависимостей
```bash
pip install -r requirements.txt

## Запуск тестов
```bash
pytest tests/ -v

## Зависимости
Файл requirements.txt должен содержать:
requests==2.31.0
pytest==7.4.0
